# DDD: Modelagem Tatica e Patterns

Projeto do curso DDD: Modelagem Tatica e Patterns.

## Stack

- Node.js
- TypeScript
- Jest
- Sequelize
- SQLite
- SWC
- TSLint

## Como executar

1) Instale as dependencias:

```bash
npm install
```

2) Rode o TypeScript (checagem de tipos):

```bash
npm run tsc -- --noEmit
```

## Como rodar os testes

```bash
npm test
```
